# ⚔️ Realm of Legends

**Voxel Biome Explorer** — Un explorateur de biomes voxel dans le style Minecraft, construit avec Three.js.

🌐 **Jouer en ligne :** [https://sebdrd1.github.io/realm_of_legends](https://sebdrd1.github.io/realm_of_legends)

## 🎮 Contrôles

- **W A S D** — Se déplacer
- **Souris** — Regarder (cliquer pour capturer)
- **Espace** — Monter
- **Shift** — Descendre
- **1-9 / Numpad** — Téléportation biome
- **Échap** — Libérer la souris

## 🎯 Le Projet

Realm of Legends est un jeu sandbox ultime fusionnant :
- **Minecraft** — Construction voxel
- **Mine Colonie** — Gestion de colonie
- **Age of Empire** — Stratégie/RTS
- **Tomb Raider** — Exploration et aventure

Avec **320+ personnages** issus de films, dessins animés et manga.

## 🛠️ Stack

- Three.js (WebGL)
- HTML5 / CSS3
- GitHub Pages
